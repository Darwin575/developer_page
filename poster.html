Digitizing Document Management: A Web‑Based Tracking System for the RICTMD of the BJMP Region 10
Cañete, E. M. F. • Lariosa, G. D. • Malacaste, F. K. • Ajon, K. L.

Abstract

This study addresses critical inefficiencies in BJMP Region 10's document routing system through the development of a comprehensive web‑based Document Tracking System (DTS). Utilizing Agile Scrum methodology, the system integrates advanced QR code technology, OTP verification, and real‑time tracking capabilities.

Comprehensive evaluation demonstrates significant improvements in accessibility, security, and process transparency, directly contributing to digital transformation initiatives in public administration and aligning with RA 11032 compliance requirements.

Problem & Objectives

Core Problem: Manual document routing at BJMP RICTMD creates operational bottlenecks, security vulnerabilities, and transparency issues, hindering efficient government service delivery.

Research Objectives

  • Develop secure web‑based DTS with QR code integration
  • Implement multi‑layer security (RBAC, OTP, CSRF)
  • Create comprehensive reporting and monitoring system
  • Enable real‑time status notifications and tracking

Methodology

Development Approach

Agile Scrum Framework with continuous BJMP stakeholder feedback integration is employed throughout the development lifecycle. Key phases include Sprint Planning, Development, Review, and Deployment.

Technology Stack

  • Frontend: HTML5, CSS3, Bootstrap, jQuery
  • Backend: PHP 7.4+, MySQL 8.0
  • Security: RBAC, OTP, CSRF Protection
  • Development Tools: XAMPP, Git Version Control

Data Collection Methods

  • Surveys: BJMP staff workflow analysis
  • Interviews: Stakeholder requirements gathering
  • SUS Testing: Usability evaluation metrics
  • System Logs: Performance and usage analytics

System Architecture

Three-Tier Architecture

  • Presentation Layer: Responsive web interface with dynamic content rendering and intuitive user experience design.
  • Application Layer: Business logic processing, authentication management, QR code generation, and intelligent routing algorithms.
  • Data Layer: MySQL database managing document metadata, user profiles, audit trails, and system configurations.

Key Components

  • Authentication Module: Multi-factor authentication with OTP
  • Document Router: Intelligent routing based on document types
  • QR Generator: Unique identifiers for document tracking
  • Notification System: Real-time status updates
